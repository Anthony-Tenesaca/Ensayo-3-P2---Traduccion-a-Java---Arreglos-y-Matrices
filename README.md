Estudiante: Anthony German Tenesaca Rosales

Carrera: Tecnologia de la Informacion "B"

Objetivo

Aplicar los conocimientos sobre arreglos (vectores) y matrices en Java mediante el desarrollo de programas que permitan almacenar, recorrer, procesar y mostrar información utilizando estructuras repetitivas.

Instrucciones

Todos los programas deben desarrollarse en Java.

Utilizar Apache NetBeans como entorno de desarrollo (IDE).

Emplear la clase Scanner para la entrada de datos cuando el ejercicio lo requiera.

Declarar correctamente todas las variables antes de utilizarlas.

Mostrar mensajes claros y comprensibles para el usuario.

Resolver todos los ejercicios propuestos en el archivo proporcionado por el docente.

Subir todos los ejercicios resueltos a un repositorio de GitHub.

Completar el formato de entrega proporcionado por el docente.

En la tarea de EVEA se debe subir única y exclusivamente el formato de entrega completamente llenado. Los archivos de código fuente (.java) no deben adjuntarse, ya que deben encontrarse en el repositorio de GitHub.

Verifique que el enlace al repositorio de GitHub incluido en el formato de entrega sea correcto y permita acceder a todos los ejercicios.

Antes de enviar la tarea, compruebe que el repositorio contenga todos los archivos solicitados y que los programas compilen y ejecuten correctamente.
